Nom du droit
==========================
copixheadings
  show    - simplement pouvoir voir la rubrique concernée
  write   - pouvoir modifier / créer des sous rubrique / supprimer des sous rubriques


Chemin de base du droit
==========================
modules|copixheadings == rubrique principale

ensuite pour les sous rubriques:
modules|copixheading|name|name|name
